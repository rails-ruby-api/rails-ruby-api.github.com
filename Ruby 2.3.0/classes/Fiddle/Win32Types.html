Module Fiddle::Win32Types

  • ext/fiddle/lib/fiddle/types.rb

Adds Windows type aliases to the including class for use with Fiddle::Importer.

The aliases added are:

  • ATOM

  • BOOL

  • BYTE

  • DWORD

  • DWORD32

  • DWORD64

  • HANDLE

  • HDC

  • HINSTANCE

  • HWND

  • LPCSTR

  • LPSTR

  • PBYTE

  • PDWORD

  • PHANDLE

  • PVOID

  • PWORD

  • UCHAR

  • UINT

  • ULONG

  • WORD
