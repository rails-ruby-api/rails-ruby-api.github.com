Class JsonCherryPickTest < ActiveSupport::TestCase

  • activesupport/test/core_ext/object/json_cherry_pick_test.rb

These test cases were added to test that cherry-picking the json extensions works correctly, primarily for dependencies problems reported in #16131. They need to be executed in isolation to reproduce the scenario correctly, because other test cases might have already loaded additional dependencies.

def test_date_as_json
  require_or_skip 'active_support/core_ext/object/json'

  expected = Date.new(2004, 7, 25)
  actual   = Date.parse(expected.as_json)

  assert_equal expected, actual
end
