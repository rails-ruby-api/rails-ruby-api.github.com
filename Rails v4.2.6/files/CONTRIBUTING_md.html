Ruby on Rails is a volunteer effort. We encourage you to pitch in. Join the team!

  • If you want to submit a bug report please make sure to follow our reporting guidelines.

  • If you want to submit a patch, please read the Contributing to Ruby on Rails guide.

  • If you want to contribute to Rails documentation, please read the Contributing to the Rails Documentation section of the aforementioned guide.

We only accept bug reports and pull requests in GitHub.

  • If you have a question about how to use Ruby on Rails, please ask the rubyonrails-talk mailing list.

  • If you have a change or new feature in mind, please suggest it on the rubyonrails-core mailing list and start writing code.

Thanks! :heart: :heart: :heart:
