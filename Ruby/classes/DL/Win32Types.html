Module DL::Win32Types

  • /Users/fschumac/.rvm/src/ruby-1.9.3-p0/ext/dl/lib/dl/types.rb

Adds Windows type aliases to the including class for use with DL::Importer.

The aliases added are:

  • ATOM

  • BOOL

  • BYTE

  • DWORD

  • HANDLE

  • HDC

  • HINSTANCE

  • HWND

  • LPCSTR

  • LPSTR

  • PBYTE

  • PDWORD

  • PHANDLE

  • PVOID

  • PWORD

  • UCHAR

  • UINT

  • ULONG

  • WORD
